# :book: 0x01. ES6 Promises.

1. Promises.
2. Async Await.
3. Throw Try.

## :wrench: Project setup.
```bash
# Create project directory and readme.
mkdir ./0x01-ES6_promise
touch ./0x01-ES6_promise/README.md

# Add node packages
npm i
```

# :computer: Tasks.

# Create task file and set execute permission.
touch ./0-promise.js
